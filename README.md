# README

Basic implementation of ActionCable with React, compiled with Webpack

# Requirements

* redis-server
* ruby 2.2 or higher
* node (tested with 5.0)

# Usage

* `bundle install`
* `npm install`
* In different processes:
* - `redis-server`
* - `rails s`
* - `npm run watch`


* Launch demo at `http://localhost:3000`
